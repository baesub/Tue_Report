## 깃허브에 스크립트 파일이 적용되도록 하는 방법
1. 같은 파일 안에 lib 폴더 생성
2. 스크립트 파일을 ctrl + 클릭하여 파일 연 후 다른 이름으로 저장 -> lib 폴더 안에 저장
3. 스크립트 파일을 상대 파일 경로로 수정
